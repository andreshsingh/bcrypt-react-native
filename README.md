# bcrypt-react-native

Native implementation of bcrypt for react-native

## Installation

```sh
npm install bcrypt-react-native
```

## Usage

```js
import BcryptReactNative from "bcrypt-react-native";

// ...

const result = await BcryptReactNative.multiply(3, 7);
```
